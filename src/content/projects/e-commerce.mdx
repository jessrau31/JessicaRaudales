Como responsable del **Repositorio A**, lideré la configuración de la infraestructura crítica:
  ✔️ **Azure Storage Account**: Almacenamiento seguro de archivos estáticos/dinámicos  
  ✔️ **Blob Container**: Organización jerárquica con políticas de acceso  
  ✔️ **Azure Queue**: Coordinación de tareas asíncronas (procesamiento de pedidos)  
  ✔️ **Integración con Terraform**: Automatización con tags y versionamiento en GitHub

**Detalles Técnicos**   
🟣 **Arquitectura Multi-repositorio**: Separación clara de responsabilidades (Storage/Apps/DB)  
🟣 **Redis Cache**: Reducción de latencia en consultas frecuentes (+40% rendimiento)  
🟣 **Mensajería Asíncrona**: Descarga de tareas intensivas usando colas Azure
